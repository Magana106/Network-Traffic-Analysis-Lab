# Network Traffic Analysis Lab
**Cybersecurity Portfolio Project by Diana Magana**

---

## 🎯 Project Overview

This is a Comprehensive network traffic analysis project demonstrating packet-level 
investigation, protocol analysis, and security threat detection using wireshark.

**Project Goal:** I developed a practical network forensics skills which are applicable 
to SOC Analysts and Network Security roles.

---

## 🛠️ Tools & Technologies

- **Primary Tool:** Wireshark 
- **Capture Tool:** tcpdump 
- **Analysis Platform:** Windows 10 
- **Sample Traffic:** Real-world PCAP files from public datasets
- **Documentation:** Notion, GitHub
